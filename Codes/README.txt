1) "Data" folder consists of Kaggle time series data for different Countries.

2) "Original data" folder consists of data from Kaggle which is in raw(uncleaned) form and we have "Data_Clean.ipynb" notebook for cleaning the same.

3) "Networked_SEIR_Model.ipynb" file consists code for obtaining data from Network Model.

4) "SEIR_ODE_lockdown.ipynb" or "SEIR_ODE_lockdown.py" code runs for Lockdown analysis on solution from ODE.
